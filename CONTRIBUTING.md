# Contributing to CryptoAlertBot

Thank you for your interest in contributing! Here's how you can help.

## Reporting Bugs

Found a bug? Contact via Telegram: @lingytm

When reporting, include:
- What you were doing
- What happened
- What you expected to happen
- Your Windows version
- Error messages if any

## Suggesting Features

Have an idea? Contact: @lingytm on Telegram

Describe:
- What feature you want
- Why it would be useful
- How it should work

## Code Contributions

1. Fork the repository
2. Create a feature branch: git checkout -b feature/your-feature
3. Make your changes
4. Test thoroughly
5. Commit: git commit -m "Add your feature"
6. Push: git push origin feature/your-feature
7. Open a Pull Request

## Code Style

- Follow PEP 8 for Python code
- Use meaningful variable and function names
- Add comments for complex logic
- Keep functions focused and small

## Testing

Before submitting:
- Test on Windows
- Check that the EXE builds correctly
- Verify Telegram integration works
- Test with different cryptocurrencies

## Questions?

Contact @lingytm on Telegram for any questions about contributing.
